# 03 — 성장 루프: 킬 기반 쿨환급, 레벨업 드래프트, 런 내 성장, 스킬 진화 (P1)

## 현재 상태 (증거)

- XP 곡선: `xpNeeded = 92 + level^1.52 × 58 + level × 22` (`js/entities/Player.js:128`). 곡선 자체는 무난.
- **레벨업 연출이 약함**: `Game.onEnemyKilled`의 levelUps 처리 — 기둥+링 이펙트, 알림, 풀 회복뿐. 전투 흐름을 바꾸는 순간이 아니다.
- **스킬 포인트는 K 메뉴에서만 사용** — 전투를 멈추고 메뉴에 들어가야 성장이 완성된다.
- **스킬 가동률 고정**: 쿨다운 4.2~15.5초 + MP 소모 (`js/data/content.js` SKILLS). 킬 수와 스킬 사용 빈도가 무관 → 물량↔파워가 서로 증폭하지 않는다.
- **Hunt에 런 내 성장 없음**: Defense만 `runMods`(웨이브당 공격/방어/스킬파워/헤이스트 복리)와 파워샤드(5웨이브마다)를 가짐. 메인 모드인 Hunt는 레벨/장비뿐.
- **질적 성장 부재**: 스킬 랭크 1~5는 수치 성장만. 빌드가 변하는 순간(진화/시너지)이 없다.

## 설계 제안

### A. 킬 → 스킬 가동률 환류 (P1, 효과 대비 구현비 최소)

"많이 잡을수록 더 자주 쏜다"를 만드는 최소 메커니즘:

- **킬 쿨환급**: 킬 1마리당 모든 액티브 스킬 쿨다운 `-0.12초`(잡몹) / `-0.35초`(정예) / `-2초`(보스). 멀티킬은 합산. `CombatSystem` 킬 훅에서 `player.cooldowns` 감산 한 줄.
- **킬 MP 회복**: 킬당 MP +1.5(잡몹) — MP가 물량전에서 스킬을 막지 않도록.
- 밸런스 가드: 스킬당 내부 최소 재사용 간격 1.2초(무한 연사 방지).
- 기대 체감: 물량 밀도(P0) 도입 시 분당 킬 40~80 → 실질 쿨다운이 절반 이하로 — "스킬을 돌리는 게임"으로 전환된다.

### B. 레벨업을 전투 안의 사건으로 (P1)

- **레벨업 노바**: 레벨업 순간 반경 4.5m 넉백 + 잡몹 즉사(정예는 큰 피해) + 1.2초 무적 + 화면 가장자리 골드 비네트 펄스. 물량 한복판에서 레벨업이 "구원"이 되게.
- **즉석 드래프트(선택)**: 레벨업 시 화면 하단에 3택 카드(1/2/3 키로 즉시 선택, 이동 가능 상태 유지) — 스킬 포인트 자동 배분 or 소형 런 버프 중 택1. K 메뉴는 유지하되 "메뉴에 안 들어가도 성장이 완성"되도록.
  - 카드 풀 예시: `Q 스킬 +1랭크`, `공격 +6%`, `픽업 반경 +25%`, `이속 +5%`, `킬 쿨환급 +0.04s`.
- 최소 구현 버전: 드래프트 없이 노바 + "스킬 포인트 자동 추천 배분(토글)"만으로도 흐름 단절 제거 가능.

### C. Hunt 런 내 성장 — 유물(Relic) 드랍 (P1~P2)

Defense의 runMods에 해당하는 층을 Hunt에 부여:

- 정예/보스/계약 보상으로 **유물** 드랍(런 한정, 저장 안 함): 픽업 즉시 발동하는 패시브. 예: "킬 체인 10+ 동안 공격 +8%", "젬 픽업 시 5% 확률 소형 노바", "대시가 지나간 자리에 화염 트레일".
- 슬롯 3개, 초과 획득 시 교체 선택(플로트 UI).
- `player.runMods` 구조는 Defense에 이미 있음 — Hunt에서도 채워 넣는 형태로 재사용.

### D. 스킬 진화 / 시너지 (P2)

- 각 스킬에 **랭크 5 도달 시 진화 선택지 2종**: 수치가 아니라 형태가 변한다.
  - 예: Whirlwind → `폭풍 지속형(이동하며 시전)` or `당김 소용돌이(적을 끌어모음 — 물량 시너지)`.
  - 예: Fireball → `3연발` or `착탄 후 화염 지대`.
- 데이터 구조: `SKILLS[id].evolutions = [{id, name, desc, apply}]`, `player.skills[id] = {rank, evolution}` — 저장 포맷 v5 마이그레이션 필요(`saveVersion` 4 → 5, 구버전은 evolution: null).
- 진화는 "물량을 더 잘 잡는 방향"의 선택지를 최소 1개씩 포함할 것(끌어모음, 관통, 연쇄, 지대).

### E. 페이싱 튜닝 (수치 목표)

- 레벨 1→10: 8~10분(현재 추정 12~15분, 밀도 상향+XP젬과 함께 재측정).
- 첫 스킬(레벨 3) 도달: 90초 이내.
- 장비 교체 빈도: Hunt에서 분당 0.5~1회(현행 `gearChance` 10.5~34%는 밀도 상향 시 상향 킬 수로 자연 증가 — 잡몹 드랍률은 별도 하향 필요: 잡몹 0.02, 정예/일반 현행).

## 수용 기준

1. 물량전 한복판에서 스킬 쿨다운이 체감상 "거의 항상 준비됨" 상태가 된다(분당 킬 50 기준).
2. 레벨업 순간이 전투를 유리하게 바꾸고, 메뉴에 들어가지 않아도 성장이 완성된다.
3. Hunt 한 판(15~20분) 동안 캐릭터가 시작 대비 명확히 다른 빌드/파워로 끝난다.
4. 저장/로드 후 진화·유물 상태가 올바르게 복원(유물은 런 한정이므로 저장 제외)된다.
