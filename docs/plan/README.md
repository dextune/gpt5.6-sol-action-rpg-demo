## When to add a plan here

Create `docs/plan/<topic>.md` only for **in-flight** multi-step work. While active:

1. Link it from [../README.md](../README.md) under **Active plans**.  
2. Keep status at the top (`draft` / `ready` / `in progress`).  
3. When integrity-green and product-accepted, **move the file to `docs/history/`** and update links (hub, `AGENTS.md`, any agent docs that pointed at it).

Do not leave completed checklists in this folder — they become stale “todo” traps for agents.
